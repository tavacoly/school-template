📝 راهنمای استفاده از قالب HTML مدرسه امید

سلام و وقت بخیر ✨
از اینکه قالب HTML "مدرسه امید" را انتخاب کرده‌اید، متشکریم.  
این قالب مناسب برای طراحی پنل مدیریتی مدرسه می‌باشد و شامل صفحات مختلف برای معلم، دانش‌آموز، مدیریت و فرم‌ها است.

─────────────────────────────

📁 ساختار پوشه‌ها:
├── index.html               → صفحه ورود
├── signup.html              → فرم ثبت‌نام
├── home.html                → داشبورد اصلی
├── profyle.html             → پروفایل دانش‌آموز
├── profyle2.html            → پروفایل معلم
├── listclass.html           → لیست کلاس‌ها
├── lisichild.html           → لیست دانش‌آموزان
├── L-personel.html          → لیست معلمان
├── L-number.html            → نمرات
├── L-message.html           → پیام‌ها
├── L-atten.html             → حضور و غیاب
├── khabar.html              → صفحه اخبار
├── chart.html               → نمودارها

├── formchild.html           → فرم اطلاعات دانش‌آموز
├── formclass.html           → فرم کلاس
├── formnews.html           → فرم اخبار
├── formaten.html            → فرم حضور و غیاب
├── formpersonel.html        → فرم اطلاعات معلم/پرسنل

├── assets/
│   ├── css/
│   │   └── style2.css        → فایل‌های استایل سفارشی
│   ├── js/
│   │   └── app.js            → فایل‌های جاوااسکریپت
│   └── images/
│       └── تصاویر (لوگو، آواتار، آیکن و...)

├── preview.html             → فایل پیش‌نمایش صفحات قالب (برای خریدار)
├── README.txt               → این فایل راهنما

─────────────────────────────

💡 تکنولوژی‌های استفاده‌شده:
- HTML5 / CSS3
- Bootstrap 5.3 RTL
- TailwindCSS 3.4
- jQuery + AJAX
- Chart.js برای نمودارها

─────────────────────────────

⚠️ نکات مهم:
- این قالب فقط ظاهر (فرانت‌اند) می‌باشد و هیچ پنل مدیریتی یا اتصال به پایگاه داده ندارد.
- فونت‌ها و استایل‌ها کاملاً راست‌چین و سازگار با زبان فارسی هستند.
- تمام صفحات ریسپانسیو هستند و روی موبایل، تبلت و دسکتاپ به‌خوبی نمایش داده می‌شوند.

─────────────────────────────
